Compounds, isotopes, dark matter

Compounds

Signal chains across elements

Multiple elements bonding into a derived signal. Ld + Mn + Cr = legislative-doc-flagged spend, discussed in minutes, check appears = the "found it first, not all" closure pattern. Sb + Mp + Ci = subdivision plat under master plan referenced by capital plan = development trajectory. Au + Pn + Ds = audit + pension + debt service = credit-health composite. Bo + Ca + Mg = bond OS + CAFR + management letter = bondholder triangulation.

Isotopes

Same element, different vendor format

Cr-Trumbull, Cr-Nashua, Cr-Bedford, Cr-Montpelier — same atomic class (check register), four different layouts. Vendor-fingerprinting (filed today as open-item) is the move that treats these as isotopes of one element rather than four separate elements. Pa-Vision-CAIDump, Pa-MUNIS-Augusta, Pa-MUNIS-Cumberland — same on the parcel side. Slice 2d work is parcel-side isotope unification.

Allotropes

Same material, different presentation

A budget can appear as Bg (the budget itself), Bt (the transmittal letter), or part of Ag (in a board packet). The underlying material is the same; the surface form determines which extractor lights up. Useful when triaging "where does this signal live?" — the answer is often "across three elements that look unrelated until you see the bond."

Dark matter — half-condensed

The document bucket: 153K → 140K, and what's left

Title-stage backfill (16,951 rows) + body-stage fallback (994 rows) reclassified 17,945 rows out of the catch-all — 153,450 → 140,541 (−8.4%). The remaining mass splits by extraction_status: ~91K rows are aged_out (PDFs the navigator's age filter rejected at discovery time — never downloaded, not text-cleared as initially assumed); ~13K have body text but match nothing in the title+body classifier; ~17K are correctly noise-filtered; ~5K need OCR.

Path A is in flight. tools/reset_aged_by_title_pattern.py reset 5,082 aged_out rows across six buyer-segment bundles (bond credit 3,381 / governance 949 / redev 449 / environmental 138 / legal 117 / intergovernmental 48) for re-download on the next pipeline pass. The fiscal bundle (8,737 rows) is deferred until disk tiers down.
